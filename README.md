# 📚 电子书阅读器

一个基于 HTML5 的本地电子书阅读应用，无需服务器，直接在浏览器中运行。

## ✨ 功能特性

### 📖 文件上传
- 支持上传本地 `.txt` 格式小说文件
- 支持上传本地音乐文件作为背景音乐（`.mp3`, `.ogg`, `.wav` 等）

### 📑 智能章节识别
自动识别多种章节格式：
- `第一章 标题`、`第1章 标题`
- `Chapter 1 Title`
- `1. 标题`、`1 标题`
- `前言`、`楔子`、`序章`、`后记`、`尾声`

### 📱 阅读功能
- 上一页 / 下一页 翻页
- 键盘快捷键支持（← → 方向键）
- 点击目录快速跳转章节
- 阅读进度自动保存
- 进度条显示

### 🎵 背景音乐
- 本地音乐文件播放
- 播放/暂停控制
- 音量调节
- 循环播放

### 🎨 阅读设置
- **主题模式**：白天 / 夜间 / 羊皮纸
- **字体大小**：12px - 32px 可调
- **数据管理**：清除所有本地数据

## 🚀 使用方法

1. **下载项目文件**
   ```
   index.html    # 主页面
   styles.css    # 样式文件
   app.js        # 应用逻辑
   ```

2. **打开应用**
   - 直接用浏览器打开 `index.html` 文件
   - 或部署到任何静态网站托管服务

3. **开始使用**
   - 点击「上传小说文件」选择 `.txt` 小说
   - （可选）点击「上传背景音乐」选择音乐文件
   - 点击「开始阅读」

4. **阅读操作**
   - 使用底部的「上一页」「下一页」按钮翻页
   - 或使用键盘 ← → 方向键
   - 点击左上角 📑 打开目录
   - 点击右上角 ⚙️ 打开设置

## 💾 数据存储

所有数据（设置、阅读进度）均保存在浏览器本地（localStorage），不会上传到任何服务器。

## 🌐 浏览器兼容性

- Chrome / Edge（推荐）
- Firefox
- Safari

## 📄 小说文件格式建议

为了获得最佳的章节识别效果，建议使用以下格式：

```
第一章 标题
章节内容...

第二章 标题
章节内容...
```

或者：

```
第1章 标题
章节内容...

第2章 标题
章节内容...
```

## ⚠️ 注意事项

1. 小说文件需为纯文本格式（`.txt`）
2. 建议使用 UTF-8 编码以避免中文乱码
3. 音乐文件大小建议在 20MB 以内
4. 阅读进度按文件名识别，重命名文件会导致进度丢失

## 🔒 隐私说明

- 所有文件均在本地处理，不会上传到服务器
- 阅读进度和设置保存在浏览器本地存储中
- 清除浏览器数据会导致阅读进度丢失
